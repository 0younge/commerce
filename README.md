#  실시간 커머스 플랫폼

간단한 콘솔 기반 커머스 시스템입니다.  
사용자는 상품을 조회하고 장바구니에 담아 주문할 수 있으며, 관리자는 상품을 추가할 수 있습니다.

---

## 주요 기능

###  사용자 기능
- 카테고리별 상품 조회
- 상품 선택 및 장바구니 추가
- 장바구니 확인
- 주문 진행 / 취소

###  장바구니 기능
- 상품 담기 (중복 시 수량 증가)
- 총 금액 계산
- 주문 시 재고 차감
- 장바구니 초기화

###  관리자 기능
- 상품 추가 (카테고리 선택 후 등록)
- 관리자 비밀번호 인증 (`admin123`)

---

## 프로젝트 구조

```
프로젝트
 ┣ 📄 Main.java              # 실행 시작
 ┣ 📄 CommerceSystem.java    # 전체 흐름 관리 (메인 시스템)
 ┣ 📄 AdminSystem.java       # 관리자 기능
 ┣ 📄 Category.java          # 카테고리 관리
 ┣ 📄 Product.java           # 상품 정보
 ┣ 📄 Basket.java            # 장바구니
 ┗ 📄 BasketProduct.java     # 장바구니 내 상품
```

---

## 실행 흐름

```
메인 메뉴
 ┣ 카테고리 선택
 ┃ ┗ 상품 선택 → 장바구니 추가
 ┣ 장바구니 확인 → 주문
 ┣ 주문 취소
 ┗ 관리자 모드
     ┗ 상품 추가
```

---

## 핵심 클래스 설명

### `Product`
- 상품 정보 관리 (이름, 가격, 설명, 재고)
- 재고 차감 기능 제공

### `Category`
- 상품 리스트 보유
- 카테고리별 상품 출력

### `Basket`
- 장바구니 리스트 관리
- 총 금액 계산 및 주문 처리

### `BasketProduct`
- 장바구니에 담긴 상품 + 수량 관리

### `CommerceSystem`
- 사용자 인터페이스 및 전체 흐름 제어

### `AdminSystem`
- 관리자 기능 (상품 추가)
